About Me

Hi, I'm Casbian—a developer and digital creator passionate about coding, game development, and artistic asset creation. My work ranges from building robust Python packages to crafting evolving games, and even designing simple but striking pixel art and Wallpaper Engine backgrounds.

I take pride in my projects like CoreEase—a powerful Python package to simplify workflows—and SoulWoods, my evolving game that blends classic mechanics with a creative GUI. I also enjoy exploring mysterious scripts like CCKnight, which challenge conventional coding with their enigmatic approach.

Projects

CoreEase

CoreEase is a robust Python package designed to automate repetitive tasks and manage threads efficiently. It offers features such as browser automation, file/directory operations, and even compiler support for turning Python scripts into executables.

  • Thread Management with animated load buffers
  • Browser Automation including headless options
  • File and Directory Operations
  • Console Utilities and System Info Retrieval
  • Compiler support to build standalone executables

View CoreEase on GitHub

SoulWoods

SoulWoods is an evolving game project built with Python's tkinter and customtkinter. It features interactive story screens, dynamic combat with animated characters, and upgrade mechanics—all powered by custom game logic.

The codebase includes modules like Monster.py, Player.py, and Upgrade.py, with the main game loop in v2.0.py. Although the game is still in development, I'm proud of the progress and creative direction it represents.
