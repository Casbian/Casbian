Software Projects

CoreEase

CoreEase is a powerful Python package designed to automate repetitive tasks and manage threads efficiently. It offers browser automation, file/directory operations, console utilities, and even compiler support for turning Python scripts into executables.

  • Thread management with animated load buffers
  • Headless browser automation
  • File and directory operations
  • System information retrieval and console utilities
  • Compiler support for building standalone executables

View CoreEase on GitHub

SoulWoods

SoulWoods is an evolving game project built with Python's tkinter and customtkinter. It features interactive story screens, dynamic combat with animated characters, and upgrade mechanics driven by custom game logic.

The project includes modules such as Monster.py, Player.py, Upgrade.py, and a main game loop in v2.0.py.
